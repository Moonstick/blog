## To create a new post

1. Copy the folder `blog/any-existing-post/` to `blog/your-new-slug/`
2. Replace the `index.mdx` content with this template
3. Set `date` to today's date in YYYY-MM-DD format
4. Set `slug` to match your folder name (lowercase, hyphens)
5. Add a hero image to the folder and update `hero_image: "./your-image.jpg"`
6. Write your content below the frontmatter
7. Commit and push — the post goes live automatically
